# что такое installer?
installer - утилита в которой находятся самые крутые

# Установка Installer на Termux

+ apt update

+ apt upgrade -y

+ pkg install python -y

+ pkg install git -y

+ git clone https://github.com/777-FOXik-777/installer

+ cd installer

+ python3 installer.py
